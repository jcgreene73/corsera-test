CSS Styling

css is styling for a website. You embed it in the HTML code or you can link it as an external file to the website. To link it, add a tag in the head section, <link rel="stylesheet" href="the css file to link.css">.

CSS is composed of elements, classes and id selectors. You specify the tag to apply to by just defining the tag name (i.e. "p" for paragraph tags, "div" for div tags). This allows you to then define arguments about styling for the tag selected. An example would be "p { color: blue; }". This would style all of the "p" elements to have blue text.

Class selectors allow you to style certain pieces only as opposed to the whole document. You add the class identifier to the html tag then you add the class styling to the css document. An example would be "<p class="red">Text that would be red</p>" in the html document, then ".red { color: red; }" in the css document. A class is ALWAYS defined with a dot before the name in the CSS document.

You can also stlye items based off of their ID in the HTML document. You do so by adding a selector of #id_name in the CSS document. An example would be "#splash { color: green; }". This will color the text of anything with the id "splash" to green.

Lastly, you can group selectors for efficiency. Add a "," after the first selector then add the next selector.

Aswell, in CSS, the last declaration before the tag wins. If you only have a CSS sheet linked, that will take precedence. If you add a style tag to an HTML tag in the HTML doc, it will take precedence over the CSS sheet.

An element with a class selector that has a CSS definition will change how the element is styled. This will add a new definition for ONLY that element when it has ONLY that class selected. You add it by using "tag.class" in the CSS document then by using "<tag class="class tag">".

You can also have a child selector, which will apply only when the child tag is present in the parent tag. An example, "article > p". This will only apply when the child is a direct child of an element

You could also have a descendant selector. In this case, any time the descendant is present, it will apply the style. An example, "article p". In this example, all "p" tags within an "article" tag will be styled, regardless of parent order.

This applies to ALL selector types.
